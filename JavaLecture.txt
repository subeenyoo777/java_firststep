
자바는 클래스와 객체로 동작한다!
    🔑 핵심
      - class: 설계도
      - new : 실물 객체 생성
      - . : 객체가 가진 메서드, 변수에 접근
      - main : 프로그램이 시작되는 곳이다.

    1. 클래스(class) 선언
        : 프로그램 틀(설계도)를 만든다.

    2. 필드, 맴버 변수
    : 클래스 안에 데이터(값)을 담아둔다

    3. 메서드(method)
        : 클래스 설계도로 실제 사용 가능한 실체(인스턴스)를 만든다

    4. 메인 메서드(main)
        : 프로그램이 시작되는 진입점

    5. ***객체를 통해 메서드를 호출한다
        : 만든 객체를 이용해 값을 넣고 기능을 실행한다.
          => 객체를 만들어야 매서드(이해한 바로는 action)을 취할 수 있다.
